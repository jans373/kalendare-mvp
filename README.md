# 🗓️ Open Source Fotokalendář MVP

Tento projekt je připraven jako základ pro webovou aplikaci na tvorbu fotokalendářů. Obsahuje frontend (React + Vite) a backend (Node.js + Express).

## 📦 Jak spustit

### 1. Frontend
```bash
cd client
npm install
npm run dev
```

### 2. Backend
```bash
cd server
npm install
node index.js
```
